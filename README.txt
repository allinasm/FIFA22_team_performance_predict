
This project aims to analyze if FIFA player's parameters are predictors of their teams sports performance in the real world.

It's composed by two notebooks:

FIFA22_merge_Transfermarkt.ipynb contains the code to prepare de dataset. Mostly is a .csv download from
https://www.kaggle.com/datasets/stefanoleone992/fifa-22-complete-player-dataset with the players stats,
and a web scraping proces from transfermarkt.

FIFA22_as_predictor.ipynb contains the feature engineering process and PCA.
